# yorugata app

夜型のあなたのためのシンプルな Todo アプリです。  
「今夜やること」をサクッと追加して管理できます。

---

## 📝 機能

- タスクの追加
- タスクの完了チェック
- タスクの削除

---

## 🚀 使い方

### 1. タスクを追加
「＋」を押すと入力欄が表示されます。  
Enterキーまたは「追加」ボタンで追加できます。

### 2. タスクを完了
チェックボックスをクリックすると完了になります。

### 3. タスクを削除
「削除」ボタンでタスクを削除できます。

---

## 🧱 技術スタック

- Next.js (App Router)
- TypeScript
- Tailwind CSS
